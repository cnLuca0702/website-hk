# 文档知识库构建系统

基于 docling + ExtractThinker + knowledge-base-builder 三层架构的文档自动识别、分类、抽取与知识库构建系统。

## 核心功能

1. **文档上传**：支持 PDF、DOCX、XLSX、PPTX 等多种格式
2. **智能分类**：基于 LLM 的文档类型自动识别（发票、合同、报告等）
3. **字段抽取**：使用 Pydantic Contract 定义抽取字段，结构化输出
4. **知识库构建**：多文档合并、LLM 摘要、结构化 Markdown 输出

## 快速开始

### 后端启动
```bash
cd backend
pip install -r requirements.txt
python main.py
```

### 前端启动
```bash
cd frontend
npm install
npm run dev
```

## 依赖的三方库

- **docling**: 文档解析（PDF/DOCX/PPTX/XLSX → Markdown/JSON）
- **extract-thinker**: 文档分类与字段抽取
- **knowledge-base-builder**: 知识库构建与合并
